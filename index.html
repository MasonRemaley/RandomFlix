RandomFlix

Download this project as a .zip file Download this project as a tar.gz file

This simple bookmarklet gives Netflix TV shows a random-episode functionality.

Installation

Drag this link to your bookmarks bar: RandomFlix

In Firefox, the bookmarks bar is not enabled by default. To enable it, right-click (or control-click) on the toolbar and then check the "Bookmarks Bar" option.

In Chrome, the bookmarks bar is not enabled by default. To enable it on windows, press Control-Shift-B. On OSX, press Command-Shift-B.

Instructions

Navigate to the Netflix show you want to watch. Click on the RandomFlix button in your bookmarks bar. That's it, have fun!
